# Alchemy Game

Однопользовательская игра на C# WinForms.  
Комбинируй базовые элементы и открывай новые — всего 30 элементов и 34 рецепта.

# Геймплей

1. Нажми на элемент в инвентаре → он появится на рабочем поле
2. Перетащи один элемент на другой
3. Если рецепт существует — получишь новый элемент
4. Открой все 30 элементов!
